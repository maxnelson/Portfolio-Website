Timeline is a web application for creating and sharing timelines.

This began as a fun, self-initiated, side-project for me back in 2017, and was something I’d spend my nights and weekends hacking on. During my recent stint of unemployment I’ve had plenty of time to focus my work on this project and the result is a very exciting web application which I believe holds a lot of potential.

The current version of the application is live and viewable at:

https://timelineapp.io

I am the sole developer of this project, and am currently dedicated to adding new features on a weekly basis.

Technologies Used: Google Cloud Platform, Firebase, Firestore, Node, Vite, Express, React, D3, OAuth2
